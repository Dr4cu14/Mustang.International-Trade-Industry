# 国际贸易全流程业务文档

## 目录结构说明

本系列文档按功能分为四个维度：

### **A_基础贸易流程** - 传统国际贸易的完整业务流程
### **B_特殊贸易模式** - 特定贸易模式的专业领域  
### **C_技术支撑体系** - 技术、标准、应急、可持续发展
### **D_人才培养体系** - 人才培养与职业发展
